Markdown Cheatsheet

标题

一般使用 n ( n ≤ 6 ) 个 # + Space + 标题内容 创建 N 级标题，或者使用 n 个 # + Space + 标题内容 + Space + n 个 #。

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

文字样式

  • 斜体 ： 斜体文字 or 斜体文字
  • 加粗 ： 加粗文字 or 加粗文字
  • 加粗斜体 ： 加粗斜体文字 or 加粗斜体文字
  • 删除线 ： 删除文字
  • 行内代码块 : 行内代码块
  • 键位标识 ： 键位名称
  • Emoji ： :Emoji Code: ，比如 :yum:
    • Emoji Code 可以参见 → emoji-cheat-sheet
- 斜体 ： _斜体文字_ or *斜体文字*
- 加粗 ： __加粗文字__ or **加粗文字** 
- 加粗斜体 ： ___加粗斜体文字___ or ***加粗斜体文字***
- 删除线 ： ~~删除文字~~
- 行内代码块 : `行内代码块`
- 键位标识 ： <kbd>键位名称</kbd>
- Emoji ： :EMOJICODE:，比如:yum:
	- Emoji Code 可以参见 → [emoji-cheat-sheet](http://emoji-cheat-sheet.com)

文字链接

- 一般链接 ： 
	- [带链接的文字内容](链接地址 "链接提示文本") 
	- 链接地址
	- <链接地址>
- 本文链接：
	- [带链接的文字内容](#标题内容 "链接提示文本")

列表

无序列表

  • 无序列表示例

  • 无序列表语法如下：

    * 无序列表项一
* 无序列表项二
	* 缩进的列表项一
	* 缩进的列表项二
		* 支持无限级别缩进
	* 可以退回上一级

有序列表

  1. 有序列表示例

  2. 有序列表语法如下：

    1. 有序列表项一
2. 有序列表项二
	* 可与无序列表穿插
	* 无序列表穿插实例
2. 标号可随便写，会自动纠正
	1. 再次缩进
		1. 继续缩进
	2. 返回上一级

任务列表

- [ ] 未完成的任务项
- [x] 已完成的任务项
	- 可以与其它列表穿插
		- [ ] 穿插的未完成任务项

代码段

使用 ``` + 语言名称 + Enter + 代码片段 + Enter + ``` ，添加代码片段并根据指定的语言进行高亮。

引用

使用 > + Space + 引用文本 创建引用样式，引用支持无限缩进。
